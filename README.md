# onlinestore
orders.csv file contains 5 orders with different order dates, products, and quantities.

The python program "online_store_analysis.py" uses the csv module to read the data from the CSV file "orders.csv", and then computes the total revenue generated by the online store for each month, by each product, and by each customer using defaultdict. It then prints the results in a human-readable format.

We can run the Python code by navigating to the folder where you saved the file using the command line and typing python online_store_analysis.py (assuming you have Python installed on your computer).
